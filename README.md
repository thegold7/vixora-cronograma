# VIXORA Cronograma 2026

Sistema de asignación de actividades para técnicos — Next.js + Google Sheets.

## Variables de entorno en Vercel

```
GOOGLE_SHEETS_ID=<tu-sheet-id>
GOOGLE_SERVICE_ACCOUNT_EMAIL=<tu-sa-email>
GOOGLE_PRIVATE_KEY="<tu-private-key-con-\n>"
VIXORA_EDITOR_PASSWORD=<tu-contraseña-editor>
```

## Stack

- Next.js 16 + TypeScript
- Tailwind CSS 4 + shadcn/ui
- Google Sheets API (Service Account)
- Zustand
